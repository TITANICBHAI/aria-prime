# ARIA — How to Use

> Practical guide for day-to-day use. How to write good goals, manage tasks, and get the best results.

---

## Writing Good Goals

ARIA understands plain English. Be specific and describe the end result, not the steps.

**Good goals:**
- `Open the Netflix app, search for Inception, and start playing it`
- `Go to my Gmail, open the most recent email from Amazon, and copy the tracking number`
- `Open the calculator and calculate 15% of 2340`

**Avoid vague goals:**
- `Do stuff on my phone` ← too vague
- `Open something` ← no target
- `Be helpful` ← not actionable

**Multi-step goals work:**
ARIA will automatically break down a long goal into individual steps. You do not need to give one instruction at a time.

---

## Using Presets

Presets are saved goals you run regularly.

**Save a preset:**
1. Type your goal in the Control screen.
2. Tap the **bookmark icon** next to the input field.
3. Give it a name (e.g. "Morning News").
4. Tap Save.

**Run a preset:**
1. Tap the presets list below the input field.
2. Tap any preset to load it instantly.
3. Tap Start.

---

## Task Queue (Running Multiple Goals)

You can queue up several goals to run in order.

1. Enter a goal and tap **Add to Queue** (instead of Start).
2. Add more goals to the queue.
3. Tap **Start Queue** — ARIA will complete each goal in order before moving to the next.

View and reorder the queue on the **Activity** screen under the Queue tab.

## General Mode vs Gaming Mode

**General mode** (default) — Works best for standard Android apps (WhatsApp, Gmail, Chrome, YouTube, etc.). ARIA uses the accessibility tree to understand the screen precisely.

**Gaming mode** — Switch to this for mobile games. Games do not expose an accessibility tree, so ARIA switches to pure vision: it takes screenshots, identifies game elements visually, and uses coordinate-based taps. Gaming mode is slower but works where General mode cannot.

**How to switch:**
Control screen → Mode selector → tap **Gaming**.

---

## Learn-Only Mode

Use this when you want ARIA to observe what you do without acting itself.

**Use case:** You are doing something complex on your phone and you want ARIA to watch and build memory about how it works — but you do not want it to interfere.

1. Switch mode to **Learn Only**.
2. Use your phone normally.
3. ARIA silently records what it sees into memory.
4. Switch back to **General** mode when you want ARIA to use what it learned.

---

## Checking What ARIA Did

After running a task, go to **Activity → Action Log** to see every step ARIA took:
- Which app it opened
- What elements it tapped
- What text it typed
- Whether each action succeeded or failed

## Common Usage Examples

### Social Media
`Open Instagram, go to my DMs, read the most recent message, and reply with a thumbs up`

### Productivity
`Open Google Docs, create a new document, title it Meeting Notes, and add today's date as a heading`

### Shopping
`Open Amazon, search for wireless earbuds under 2000 rupees, sort by customer reviews, and open the top result`

### Utilities
`Set a 25-minute timer in the Clock app`

### Entertainment
`Open Spotify, go to my liked songs, shuffle play`

## Tips for Best Results

- Run tasks when your phone screen is unlocked and visible.
- Close apps you do not need — more RAM means faster inference.
- If ARIA gets stuck, tap **Stop** and rephrase the goal more specifically.
- In Gaming mode, make sure the game is already open before giving ARIA a goal.
- Use **Learn Only** mode first when trying a new app you have never used ARIA with before.
